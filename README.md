## 🎮 About YOGAME

YO is not just a data tool — it's also your "data playground."

We're building a set of mini-games that automatically generate personal data reports after each session — reaction time, decision patterns, luck streaks, and more. All data syncs directly to your YO workspace for deeper analysis.

**YOGAME is planned for Phase 3. Stay tuned.**
